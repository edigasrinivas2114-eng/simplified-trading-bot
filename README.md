# Simplified Trading Bot (Safe Mode)

## Overview
This is a simplified crypto trading bot built using Python.
Due to limited availability of Binance Futures Testnet APIs,
the bot runs in SAFE (dry-run) mode while preserving full
trading logic and structure.

## Features
- Command-line interface (CLI)
- Market and Limit orders
- Buy and Sell support
- Input handling
- Logging of all activity
- Modular code structure

## How to Run
```bash
python bot.py
